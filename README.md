# SVG Animation Collection

Welcome to the SVG Animation Collection repository! This project showcases various SVG animations that can be used in web applications and projects.

## Documentation

This collection consists of various SVG animations designed to enhance the visual appeal of your web applications. Each animation is stored in its own folder and can be easily integrated into your projects.

## Contribution Guidelines

We welcome contributions to this project! To contribute:
1. Fork the repository.
2. Create a new branch for your feature or bug fix.
3. Make your changes and commit them with clear messages.
4. Push your branch back to your fork.
5. Open a pull request and describe your changes clearly.

## Setup Instructions

To get started with the SVG animations in this collection:
1. Clone the repository:
   ```bash
   git clone https://github.com/Hafsiya/animation.git
   cd animation
   ```
2. Open the `index.html` file in your browser to see the animations in action.
3. For any modifications, feel free to edit the SVG files and refresh your browser to view changes.
